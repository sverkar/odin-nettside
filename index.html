JPGs

JPG images are good for photos, handles large color palletes without big increase in file size

GIFs

GIFs are useful for animation, but otherwise pretty bad for transparent pixels and colors

PNGs

PNG funker greit til bilder, men tar større plass. Funker ikke så bra til animasjoner.
Brukes mest til diagrammer og ikoner, siden det er bra på gjennomsiktig

SVG

Er egentlig best, god kvalitet og kan skalere uten tap av kvalitet.
